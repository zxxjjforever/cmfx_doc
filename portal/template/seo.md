# seo优化
全站seo:
```php
{$site_seo_title}               <!--SEO标题-->
{$site_seo_keywords}            <!--SEO关键字-->
{$site_seo_description}         <!--SEO描述-->
```

文章列表页：
```php
{$seo_title}                     <!--SEO标题-->
{$seo_keywords}                  <!--SEO关键字-->
{$seo_description}               <!--SEO描述-->
```

文章页：
```php
 {$post_title}    <!-- 文章标题-->
 {$post_keywords} <!--关键字-->
 {$post_excerpt}  <!-- 文章摘要-->
```

页面：
```php
 {$post_title}    <!-- 文章标题-->
 {$post_keywords} <!--关键字-->
 {$post_excerpt}  <!-- 文章摘要-->
```
